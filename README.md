## 🛡️ How it Works

The software manages browser automation tasks in the background. It uses three distinct systems to ensure reliability:

- SearXNG: This engine aggregates search results from multiple sources to provide a neutral perspective.
- Camoufox: This browser engine mimics human interaction to navigate sites without triggering alarms.
- CloakBrowser: This system modifies your footprint to reach protected websites.

These components work together to provide clean, structured data to your AI tools. This reduces the chance of errors, often called hallucinations, because the AI reads live text from the page rather than relying on outdated internal memory.

## 🔧 Troubleshooting

If you encounter issues, check these common fixes:

- Connection Timeouts: Ensure that no other program uses the port assigned to browser-search. You can change this port in the settings menu.
- Blocked Requests: If a website blocks your agent, toggle the CloakBrowser setting in the options panel. This refreshes your browser fingerprint.
- Slow Performance: Check your internet connection. Large pages take longer to process than small text files.
- Updates: Keep the application current. Check the release page every few weeks for improvements to the browsing engines.
